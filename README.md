# Server Performance Stats

This project is a Bash script that analyzes basic Linux server performance statistics.

## Features

- Total CPU usage
- Total memory usage
- Total disk usage
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- OS version
- Uptime
- Load average
- Logged-in users
